**Next.js Project**

This is a Next.js project bootstrapped with create-next-app.

# Getting Started
First, install the dependencies:
npm install
or
yarn install

# Run the development server:
npm run dev
or
yarn dev

# Open your browser and visit:
http://localhost:3000

You will see your Next.js application running.

# Editing the Project
You can start editing the main page by modifying:
app/page.tsx

The page will automatically update as you save your changes because of Next.js Fast Refresh.

# Fonts
This project uses next/font to automatically optimize and load Geist, a modern font family created by Vercel.

# Script
npm run dev – Starts the development server
npm run build – Builds the production application
npm start – Runs the production build

# Technologies Used
Next.js
React
TypeScript
next/font
Geist font by Vercel
